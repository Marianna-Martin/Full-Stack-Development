## List of some basic tools


````markdown
### Basic HTML Elements and Their Functions

1. `<!DOCTYPE>`: Declares the HTML version being used in the document.

2. `<html>`: Defines the root element of an HTML page.

3. `<head>`: Contains meta-information about the document, such as title, styles, scripts, and meta tags.

4. `<title>`: Sets the title of the document, which appears in the browser's title bar or tab.

5. `<meta>`: Provides metadata about the HTML document, such as character set, viewport settings, and author information.

6. `<body>`: Contains the visible content of the document, including text, images, links, and other elements.

7. `<h1>`, `<h2>`, `<h3>`, ... `<h6>`: Define headings of different levels, where `<h1>` is the highest level and `<h6>` is the lowest.

8. `<p>`: Represents a paragraph of text.

9. `<a>`: Creates hyperlinks to other web pages or resources.

10. `<img>`: Embeds images into the document.

11. `<ul>`: Defines an unordered list, typically displayed with bullet points.

12. `<ol>`: Defines an ordered list, typically displayed with numbers or letters.

13. `<li>`: Represents a list item within `<ul>` or `<ol>`.

14. `<div>`: Defines a division or section in an HTML document, often used for layout purposes.

15. `<span>`: Defines a section of text within another element, often used for styling or scripting purposes.

16. `<br>`: Inserts a single line break within text.

17. `<hr>`: Represents a thematic break or horizontal rule, typically displayed as a horizontal line.

18. `<table>`: Defines a table for organizing data into rows and columns.

19. `<tr>`: Defines a row within a table.

20. `<td>`: Defines a cell within a table row.

21. `<th>`: Defines a header cell within a table row.

22. `<form>`: Creates a form for collecting user input.

23. `<input>`: Represents an input control, such as a text field, checkbox, or radio button, within a form.

24. `<textarea>`: Creates a multiline text input control within a form.


## 























25. `<button>`: Defines a clickable button.

26. `<pre>`: Represents preformatted text, preserving spaces and line breaks as they appear.
````
